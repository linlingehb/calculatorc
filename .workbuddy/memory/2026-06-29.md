# 2026-06-29 工作日志

## 科学计算器项目创建

- 基于 Kivy 框架创建了完整的科学计算器应用 (`main.py`)
- 支持功能：基本四则运算、三角函数(sin/cos/tan)、对数(log/ln)、幂运算(x²/xʸ)、平方根(√)、阶乘(n!)、倒数(1/x)、百分比(%)、常数(π/e)、正负切换(±)
- DEG/RAD 模式切换：求值引擎自动包裹 `math.radians()` 进行度数转换
- UI 使用 5 列 GridLayout 布局，颜色区分数字/运算符/函数按钮
- Python 版本：3.13.12 / Kivy 2.3.1
